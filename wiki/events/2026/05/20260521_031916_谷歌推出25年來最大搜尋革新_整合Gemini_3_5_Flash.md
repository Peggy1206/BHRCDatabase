# 谷歌推出25年來最大搜尋革新，整合Gemini 3.5 Flash

**Category**: Events
**Tags**: AI搜尋, Gemini 3.5 Flash, 谷歌, 搜尋引擎革新, 多模態AI, 智能代理, 市場滲透, 科技競爭, Anthropic投資
**Ingested**: 2026-05-21 03:19 UTC

---

## Summary

谷歌在I/O大會宣布搜尋引擎的重大升級，將Gemini 3.5 Flash整合至AI Mode，提供多模態搜尋、後續提問功能，以及智能代理能力。該升級已達到10億月活用戶，查詢量每季翻倍成長，反映AI驅動搜尋在市場中的快速滲透。

## Bruce's Insight

- 鎖定中小企業 (Small Business) 工作流市場：在 Ramp Index 顯示其付費企業採用率拿下冠軍 (34.4%) 後，Anthropic 官方表示將加速把 MCP 協議工具鏈下放到中小企業市場，主打免工程師背景、靠 Claude Code 就能全自動完成辦公室文檔與知識庫管理。
- 3.5吉瓦 (GW) 算力與晶片擴展協議：Anthropic 聯合 Google、Broadcom（博通）共同宣布了一項高達 3.5吉瓦（Gigawatts） 的 TPU 算力部署計劃。此舉被市場解讀為 Anthropic 為了讓 Claude 4.0 (Mythos) 擺脫對傳統 GPU 的依賴，全面邁向自主晶片供應鏈的重要一步。

## Structured Analysis

## 核心產品升級
- **Gemini 3.5 Flash** 成為全球Gemini應用與Search AI Mode的預設模型
- 在編碼與代理基準測試上超越Gemini 3.1 Pro
- 速度比其他前沿模型快4倍，成本不到一半

## 功能擴展
- 多模態搜尋：支援文字、圖像、檔案、影片、Chrome分頁輸入
- 後續提問能力：用戶可在搜尋中追問
- 「資訊代理」：背景持續推理的AI代理，可監控租賃列表等動態資訊
- 個人智能擴展至近200個國家，支援Gmail、Google Photos等應用連接

## 市場表現
- AI Mode月活用戶超過10億
- 查詢量每季翻倍成長（過去一年）
- McKinsey報告：2025年50%消費者已使用AI搜尋，預計2028年達75%

## 競爭動態
- 谷歌對Claude製造商Anthropic投資高達400億美元
- 協議包括3.5GW張量處理單元容量與5年5GW雲端運算容量

## 產業意涵
- 搜尋引擎正經歷典範轉移，從關鍵字匹配向AI推理轉變
- 代理能力與個人化成為競爭焦點
- 大型科技公司透過投資競爭對手來鞏固生態位置
