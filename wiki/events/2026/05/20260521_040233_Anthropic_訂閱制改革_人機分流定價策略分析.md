## Summary

Anthropic 自 6/15 起將訂閱額度拆分為「人機互動」與「AFK 自動化工作流」兩層，重度自動化用戶的可用算力從等值約 $5,000 API 費用驟降至 $200，實質削減高達 96%。此舉被解讀為 Anthropic 刻意將訂閱制重新定義為「人在螢幕前的互動使用」，並將機器自動化工作負載導向按量計費的 API 定價模式。此政策對「Harness Engineering」從業者造成直接衝擊，並引發用戶向 Codex（OpenAI）遷移的競爭效應。

## Structured Analysis

## Anthropic 訂閱制改革深度分析

### 🔑 核心政策變動
- **生效日期**：2025 年 6 月 15 日
- **變動內容**：訂閱額度拆分為兩層
  - **人機互動層**（Human-in-the-loop）：如 Claude Code，維持原有額度
  - **AFK 自動化層**（Away From Keyboard）：獨立月度點數
    - Pro 方案：$20
    - Max 5x 方案：$100
    - Max 20x 方案：$200

### 📊 實際衝擊量化
- 重度 Max 20x 用戶過去透過 AFK 運行消耗的算力，社群估算等值約 **$5,000 API 費用/月**
- 新制後上限僅 **$200**，實質削減幅度為 **96%**（非官方宣稱的 40%）
- 此落差揭示官方溝通與用戶實際體驗之間的嚴重資訊不對稱

### 🎯 Anthropic 的戰略意圖
- **非單純成本控制**：與 4 月關閉 OpenClaw 事件邏輯一致
- **重新定義訂閱制邊界**：訂閱 = 人在螢幕前的互動使用，非機器自動化
- **商業模式分流**：
  - 個人/互動使用 → 訂閱制（固定月費）
  - 企業/自動化使用 → API 按量計費（可擴展但成本可控）
- **核心訊號**：Anthropic 正在為 AI Agent 規模化使用建立獨立的商業框架

### ⚠️ 廠商依賴風險（Vendor Lock-in）
- **案例**：Matt Pocock（Claude Code 最暢銷付費課程作者）公開表達強烈不滿
  - 原話：*「我從未經歷過如此令人沮喪的使用條款不透明」*
  - 行動：將所有學員導向競品 Codex（OpenAI）
- **啟示**：深度依賴單一 AI 平台的教育者、企業、工具開發者，面臨「供應商一夜改規則，整個工具鏈即刻失效」的脆弱性

### 🔧 對「Harness Engineering」的衝擊
**Harness Engineering 定義**：將 AI 轉化為「可工程化、可排程的自動化工作流」的新興技術紀律

**受影響的核心能力包括**：
- 批次任務執行
- 自動化 Issue 讀取與 PR 建立
- 多輪程式碼審查
- 背景研究自動化
- 多 Agent 並行處理
- CI/CD 管線中的自動 Bug 修復

**衝擊本質**：這些能力的經濟可行性在訂閱制框架下大幅降低，迫使從業者重新評估成本結構

### 🏁 競爭格局影響
| 維度 | 觀察 |
|------|------|
| 用戶遷移 | Matt Pocock 等 KOL 公開轉向 Codex，帶動社群跟進 |
| Anthropic 壓力 | 重度用戶流失可能影響品牌聲譽與生態系黏著度 |
| OpenAI 機會 | Codex 承接遷移用戶，強化 AI 自動化工作流市場地位 |
| 行業標準 | 此定價模式若成功，可能成為 AI 訂閱制的行業參考範本 |

### 💼 對獵頭/人才市場的潛在意涵
- **新興職能風險**：以 Claude AFK 能力為核心的「AI 自動化工程師」職能定義需重新校準
- **企業採購決策**：HR/IT 採購 AI 工具時，需將「自動化工作負載成本」納入 TCO（總擁有成本）評估
- **人才技能轉移**：Harness Engineering 人才需具備跨平台遷移能力，單一平台深度專精的風險上升

## Deepening Questions
